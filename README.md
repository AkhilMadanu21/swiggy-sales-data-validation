# Swiggy Sales Data Validation Project

## 📊 Overview
This project focuses on validating and cleaning Swiggy sales data to ensure high data quality for analysis.

## 🛠 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

• Removing duplicates  
• Data validation checks  
• Basic exploratory analysis
